# Туристический сайт города Sanya

**Тема:** Туристический сайт города.

Небольшой многостраничный сайт-гид по городу Санья (остров Хайнань): достопримечательности, кафе и рестораны, полезные приложения и контакты. Проект сделан на HTML и CSS, с единой навигацией и общим визуальным стилем.

## Страницы
- `index.html` — главная страница
- `pages/places.html` — достопримечательности (полноэкранные секции с фоном)
- `pages/food.html` — кафе и рестораны
- `pages/apps.html` — приложения и VPN
- `pages/contacts.html` — контакты

## Технологии
- HTML5 (семантическая разметка)
- CSS3 (архитектура из 4 файлов: `reset.css`, `base.css`, `layout.css`, `components.css`)
